# Smart Bill Splitter

A single-purpose website that solves a small but annoying problem:
splitting a bill among friends with tip + rounding handled automatically.

## Features

- Enter total bill amount (with or without tip)
- Optional tip percentage
- Equal split among N people
- Exact per-person and rounded per-person amount
- Shows rounding leftover (extra / short)
- WhatsApp-ready text you can copy and send to your group
- Pure HTML, CSS, and JavaScript â€“ no backend

## How to Run

Just open `index.html` in any modern browser.

## Tech Stack

- HTML5
- Vanilla CSS (no frameworks)
- Vanilla JavaScript

## Single Purpose

The site does only one thing: **splits a bill cleanly**
with helpful rounding logic and a shareable text summary.
